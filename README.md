# 🏥 Projeto GeHealth

## 🎯 Objetivo
O sistema **GeHealth** é voltado para facilitar a gestão de consultas entre 
pacientes e profissionais de saúde. Com cadastros em banco de dados, foco na experiência do 
usuário e acessibilidade, busca eliminar o uso de planilhas para 
armazenar dados e fluxos manuais da empresa, centralizando as 
operações. 🗂️📊
